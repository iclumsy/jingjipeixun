新版原则
这次不做分步骤，只把单页变清楚
不做向导 用户信息量不大，分步骤会多一次理解成本，也会让“提交”显得离得更远。
保留连续填写 用户从上到下填完即可；顶部告诉他还差什么，底部一直能提交。
压缩字段高度 姓名/性别、手机号/文化程度、学校/专业这些短字段并排，减少滚动。
附件只做提醒 附件上传功能不变，只增加“还差哪几项”的清单提示。
现有功能保留
按当前小程序真实流程映射
培训类型/报名类型特种设备、特种作业、新考证、复审全部保留。
作业类别/操作项目继续使用现有 picker 和自动填入单一操作项目逻辑。
基本字段姓名、性别、文化程度、身份证、手机号、学校、专业、单位名称、单位地址全部保留。
动态附件仍按后台附件配置和报名类型决定上传项。
协议与订阅协议勾选、隐私弹窗、订阅消息请求、提交逻辑都不改变。
驳回编辑驳回原因、重新编辑、禁止切换培训项目照旧保留。
